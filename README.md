# flashcards
Command-line flashcard app

## Usage


```bash
fc learn french.yaml

je suis Adrian
> I am Adrian
```

```bash
$ fc stats

# french.yaml
Words: 104
In-progress: 25
Learned: 12

# spanish.yaml
Words: 40
In-progress: 12
Learned: 20
```

To remove a file
```bash
$ fc forget french.yaml
Consider french.yaml forgotten.
```
